## 5. Responsive Spacing Patterns

**Format:** Tailwind Responsive Classes  
**Origin:** 🎨 Tailwind CSS

### Common Responsive Patterns

#### Main Content Padding
```tsx
// App.tsx
className="px-4 sm:px-6 md:px-12 lg:px-16"
```
- **Mobile (default):** `px-4` (16px)
- **Small (sm):** `px-6` (24px)
- **Medium (md):** `px-12` (48px)
- **Large (lg):** `px-16` (64px)

#### Main Content Vertical Padding
```tsx
// App.tsx
className="py-4 md:py-12"
```
- **Mobile (default):** `py-4` (16px)
- **Medium (md):** `py-12` (48px)

#### Main Content Bottom Padding
```tsx
// App.tsx
className="pb-24 md:pb-20"
```
- **Mobile (default):** `pb-24` (96px)
- **Medium (md):** `pb-20` (80px)

#### Sidebar Offset
```tsx
// App.tsx
className="ml-0 md:ml-16"
```
- **Mobile (default):** `ml-0` (0px)
- **Medium (md):** `ml-16` (64px) - Sidebar width offset

## 6. Usage Statistics

### Most Frequently Used Spacing Values

#### Padding
1. **`p-4`** (16px) - Very High - Standard container padding
2. **`px-4`** (16px) - Very High - Standard horizontal padding
3. **`py-4`** (16px) - Very High - Standard vertical padding
4. **`p-6`** (24px) - High - Large containers
5. **`px-6`** (24px) - High - Large horizontal padding
6. **`px-12`** (48px) - High - Responsive medium breakpoint
7. **`px-16`** (64px) - High - Responsive large breakpoint
8. **`p-3`** (12px) - High - Cards, modals

#### Margin
1. **`mx-auto`** (auto) - Very High - Center containers
2. **`mb-2`** (8px) - High - Standard bottom margin
3. **`mb-4`** (16px) - High - Standard bottom margin
4. **`mt-4`** (16px) - High - Standard top margin
5. **`ml-16`** (64px) - High - Responsive sidebar offset
6. **`mb-12`** (48px) - High - Section spacing

#### Gap
1. **`gap-2`** (8px) - Very High - Standard gap (most common)
2. **`gap-4`** (16px) - Very High - Standard gap (very common)
3. **`gap-3`** (12px) - High - Large gap
4. **`gap-6`** (24px) - High - Large gap

### Spacing Scale Summary

The project uses a consistent spacing scale based on Tailwind's default scale:
- **Base unit:** 0.25rem (4px)
- **Most common values:** 4px, 8px, 12px, 16px, 24px, 32px, 48px, 64px
- **Responsive breakpoints:** sm (640px), md (768px), lg (1024px)

---

## 📝 Notes

- All spacing values follow Tailwind CSS conventions
- No custom CSS variables for spacing (unlike colors, typography, and borders)
- Responsive spacing is extensively used for mobile-first design
- The most common spacing values are 8px, 16px, 24px, and 48px
- Gap is primarily used in flexbox and grid layouts for consistent spacing between items
