Border Radius CSS Variables shadcn/ui

Base
--radius (Base)
Value: 8px
Usage: Buttons, tooltips, containers
Origin: 📦 shadcn/ui
Small
--radius-sm (Small)
Value: 4px (calc: 8px - 4px)
Usage: Small elements
Origin: 📦 shadcn/ui
Medium
--radius-md (Medium)
Value: 6px (calc: 8px - 2px)
Usage: Medium elements
Origin: 📦 shadcn/ui
Large
--radius-lg (Large)
Value: 8px (same as base)
Usage: Large elements
Origin: 📦 shadcn/ui
Extra Large
--radius-xl (Extra Large)
Value: 12px (same as card)
Usage: Extra large elements
Origin: 📦 shadcn/ui
Card
--radius-card
Value: 12px
Usage: Cards and larger containers
Origin: 📦 shadcn/ui
Pill
--radius-button (Pill)
Value: 9999px (fully rounded)
Usage: Pill-shaped buttons
Origin: 📦 shadcn/ui
